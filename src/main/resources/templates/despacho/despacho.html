Datos del Despacho

Las ubicaciones se consultan automáticamente según el producto seleccionado.
Seleccione una ubicación para conocer el stock disponible.

Lógica aplicada

Paso Acción Descripción
1 Seleccionar producto Se consulta en qué ubicaciones existe stock.
2 Seleccionar ubicación Se muestra la cantidad disponible en esa ubicación.
3 Validar cantidad El sistema impide retirar más unidades de las disponibles.
4 Actualizar inventario Se descuenta la ubicación y se recalcula el stock total del producto.
